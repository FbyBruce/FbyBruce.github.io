安装新版 PowerShell

在 Microsoft Store 中安装新版 PowerShell

安装包管理工具 Scoop

设置 Scoop 安装路径的环境变量

先创建对应文件夹，再设置环境变量

重启电脑让环境变量生效

https://scoop.sh/#/

安装 Scoop

在新版 PowerShell 中执行官网给出的安装命令：

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression

安装 oh-my-posh

https://ohmyposh.dev/

在 Scoop 官网找oh-my-posh 的安装命令：

安装并应用 Nerd 字体

oh-my-posh 中的主题的 Icon 需要 Nerd Font 字体的支持。

从 Nerd Fonts 字体下载 列表下载任意字体，然后完成字体的安装（下载下来是字体压缩包，解压后直接将解压文件里的字体系列全都执行安装操作即可）。

也可以使用 Scoop 来安装 Nerd Font 字体（比如选择JetBrainsMono Nerd Font）

如果本地开启了梯子还是无法添加、访问到 GitHub 仓库，可以给 Scoop 配置代理：

# 在本地打开梯子，ip地址就是127.0.0.1，端口是梯子开启的端口，比如我的是7897
scoop config proxy 「127.0.0.1:7897」

结束后可以用如下命令取消代理设置：

scoop config rm proxy

字体安装完成以后，需要将字体应用到 Windows Terminal 显示项中。

配置 oh-my-posh 主题

查看 oh-my-posh 主题

https://ohmyposh.dev/docs/themes

在 PowerShell 中执行以下命令：

# blue-owl 是主题名，换成想要的主题名即可
oh-my-posh init pwsh --config ~/「blue-owl」.omp.json | Invoke-Expression

再次打开恢复了原来的样子，先打开 PowerShell 的配置文件

code $PROFILE # 使用默认编辑器打开配置文件
# 或
notepad $PROFILE # 使用windows系统的文本编辑器打开配置文件

将下面的语句复制到配置文件中，记得修改为自己喜欢的主题

# 使用 oh-my-posh 主题
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH/blue-owl.omp.json" | Invoke-Expression
